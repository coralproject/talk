## After first setup 
this 
```http://localhost:3000/admin/configure```
is the preferred route for configuration.


 If after setup Admin you can't login from ```localhost:3000``` and you get a spinner symbol, then got to ```localhost:300/admin/configure```. At “Tech Settings” you can set the “Permitted Domains” for development to ```localhost:3000```.
