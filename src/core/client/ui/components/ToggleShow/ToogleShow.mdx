# ToggleShow
A render props component to manage visible state.

## Basic usage
<Playground>
  <ToggleShow>
    {({ toggleShow, show }) => (
      <div>
        {show && <div>SHOW ME</div>}
        <button onClick={toggleShow}>ClickMe and I disappear</button>
      </div>
    )}
  </ToggleShow>
</Playground>
