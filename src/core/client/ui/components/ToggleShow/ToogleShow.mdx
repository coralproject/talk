# ToggleShow
A render props component to manage visible state.

## Basic usage
<Playground>
  <ToggleShow>
    {({ toggleShow, show }) => (
      <Flex itemGutter direction="column">
        {show && <div>Hello World!</div>}
        <button onClick={toggleShow}>Toggle Show</button>
      </Flex>
    )}
  </ToggleShow>
</Playground>
