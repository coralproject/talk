# NoScroll

Disables scrolling of `<body>`.

## Basic usage

<Playground>
  <Container
    state={{ value: false }}
    render={props => (
      <div>
        <NoScroll active={props.value} />
        <Button
          variant="ghost"
          color="primary"
          active={props.value}
          onClick={() => props.setValue(!props.value)}
        >
          Disable Scrolling
        </Button>
      </div>
    )}
  />
</Playground>
