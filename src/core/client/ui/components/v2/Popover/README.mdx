# Popover

`Popover` renders a popover dialog attached to another `Element`.

## Basic usage

<Playground>
  <Popover
    id="my-popover-1"
    body={() => <h2>This is the body</h2>}
  >
    {({ toggleVisibility, ref }) => (
      <Button
        onClick={toggleVisibility}
        ref={ref}
        variant="filled"
        color="primary"
      >
        Click me!
      </Button>
    )}
  </Popover>
</Playground>

#### Example with `placement=top`

<Playground>
  <Popover
    id="my-popover-2"
    placement="top"
    body={({ toggleVisibility }) => (
      <Flex itemGutter="half">
        <h2>This is the body</h2>
        <Button onClick={toggleVisibility} size="small">
          <ButtonIcon>close</ButtonIcon>
        </Button>
      </Flex>
    )}
  >
    {({ toggleVisibility, ref }) => (
      <Button
        onClick={toggleVisibility}
        ref={ref}
        variant="filled"
        color="primary"
      >
        Click me!
      </Button>
    )}
  </Popover>
</Playground>

## Tooltip

    dark
