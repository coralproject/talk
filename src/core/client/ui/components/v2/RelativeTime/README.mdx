# RelativeTime

Renders relative time until given `date`.

## Basic usage
<Playground>
  <RelativeTime date="2018-07-04T12:14"
    formatter={(value, unit, suffix, timestamp) => "".concat(value, " ", unit, " ", suffix)} />
</Playground>
